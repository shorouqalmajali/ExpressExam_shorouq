<p>Salary model</p>
<p>id: PK, autogenerated</p>
<p>user_id: FK, references the id field in the user model</p>
<p>salary: integer</p>
<p>effective_date: date</p>
<p>Create: creates salary for a spicific user</p>
<p>&bull; Method: Post</p>
<p>&bull; URL: /salary/4/create</p>
<p>&bull; Request body:</p>
<p>{</p>
<p>&nbsp;&quot;salary&quot;: 500,</p>
<p>&nbsp;&quot;effective_date&quot;:&quot;01-01-2021&quot;</p>
<p>}</p>
<p>&bull; Success response:</p>
<p>{</p>
<p>&nbsp;&quot;status&quot;: &quot;success&quot;,</p>
<p>&nbsp;&quot;salary&quot;: {</p>
<p>&nbsp;&quot;user_id&quot;:1</p>
<p>&nbsp;&quot;salary&quot;: 500,</p>
<p>&nbsp;&quot;effective_date&quot;:&quot;01-01-2021&quot;</p>
<p>&nbsp;}</p>
<p>}</p>
<p>&bull; Error response:</p>
<p>{</p>
<p>&nbsp;&quot;status&quot;: &quot;error&quot;,</p>
<p>&nbsp;&quot;message&quot;: &quot;A user with this id doesn&rsquo;t exists.&quot;</p>
